# BusinessClaw

Персональный ИИ-ассистент для бизнеса на базе openclaw. Позволяет анализировать факторы успешности продаж, делать предсказания и получать уведомления о критических ситуациях

## Архитектура

<img width="1659" height="778" alt="image" src="https://github.com/user-attachments/assets/3834a987-5264-4b6f-b630-e763329180ae" />

ИИ агент использует google sheets как простую базу данных. Проверки критических показателей реализованы через периодические задачи планируемые встроенными в OpenClaw средствами для `cron` задач. 
Нативный телеграм бот поддерживает кастомные команды через записи в openclaw.json и инструкции для агента по исполнению. Конфигурация осуществляется в человеко-читаемом формате. Критические рассчёты 
реализуются скриптом на Python. Поддерживается логирование. Установка может быть выполнена для существующих деплоев OpenClaw а так же с развёртыванием расширенного контейнерного образа

## Google Sheets

Данные храняться в следующем формате:

| Column           | Type    | Description                                                                 |
|------------------|---------|-----------------------------------------------------------------------------|
| `дата`           | Date    | YYYY-MM-DD. One row per day. Missing days are fine.                         |
| `выручка`        | Number  | Total revenue for the day. May be empty for future dates.                   |
| `количество лидов`| Number  | New leads.                                                                 |
| `количество продаж`| Number  | Closed sales.                                                             |
| `конверсия`      | Number  | Optional - can be calculated as продажи / лиды.                             |
| `сумма допродаж` | Number  | Total upsell revenue (or quantity – see note below).                        |
| `средний чек`    | Number  | Average amount. Can be calculated as выручка / продажи.                     |
| `план продаж`    | Number  | Target revenue for the day                                                  |
| `заметка`        | Text    | Optional - notes about the day, you are free to append                      |


## Логирование

ИИ агент записывает свои действия в LOG.md как указано в LOGGING.md    
Так же предоставляется прокси для API моделей (совместимых с OpenAI) которое позволяет записывать каждый запрос и ответ на уровне HTTP и диалога

## Бот

По умолчанию не проходит стандартную процедуру опроса пользователя а сразу переходи к делу. Обрабатывает любые инструкции для агента  
Реализует команды:  
/summary - Показать сводку по текущему состоянию   
/check - Провести проверку критических показателей
/help - Список команд

Команды отображаются в подсказках
