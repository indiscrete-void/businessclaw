# BusinessClaw

Персональный ИИ-ассистент для бизнеса на базе openclaw. Позволяет анализировать факторы успешности продаж, делать предсказания и получать уведомления о критических ситуациях

## Архитектура

<img width="1719" height="621" alt="image" src="https://github.com/user-attachments/assets/82bab6a1-a152-453a-b137-d7ee6a95b388" />

ИИ агент использует google sheets как простую базу данных. Проверки критических показателей реализованы через периодические задачи планируемые встроенными в OpenClaw средствами для `cron` задач. 
Нативный телеграм бот поддерживает кастомные команды через записи в openclaw.json и инструкции для агента по исполнению. Конфигурация осуществляется в человеко-читаемом формате. Критические рассчёты 
реализуются скриптом на Python. Поддерживается логирование. Установка может быть выполнена для существующих деплоев OpenClaw а так же с развёртыванием расширенного контейнерного образа

## Google Sheets

Данные храняться в следующем формате:

| Колонка            | Тип     | Описание                                                                 |
|--------------------|---------|--------------------------------------------------------------------------|
| `дата`             | Дата    | ГГГГ-ММ-ДД. Одна строка на день. Пропущенные дни допустимы.              |
| `выручка`          | Число   | Общая выручка за день. Может быть пустой для будущих дат.                |
| `количество лидов` | Число   | Новые лиды (заявки).                                                     |
| `количество продаж`| Число   | Закрытые продажи.                                                        |
| `конверсия`        | Число   | Необязательно — может быть рассчитана как продажи / лиды.                |
| `сумма допродаж`   | Число   | Общая сумма дополнительных продаж (или количество — см. примечание ниже).|
| `средний чек`      | Число   | Средняя сумма. Может быть рассчитана как выручка / продажи.              |
| `план продаж`      | Число   | Целевая выручка на день.                                                 |
| `заметка`          | Текст   | Необязательно — заметки о дне, вы можете добавлять свои.                 |

## Логирование

ИИ агент записывает свои действия в LOG.md как указано в LOGGING.md    
Так же предоставляется прокси для моделей (совместимых с API OpenAI) которое позволяет записывать каждый запрос и ответ на уровне HTTP

## Бот

Обрабатывает любые инструкции для агента  
Реализует команды:  
/summary - Показать сводку по текущему состоянию   
/check - Провести проверку критических показателей
/help - Список команд

Команды отображаются в подсказках

## Ограничения 

Требует мощную, мыслящую модель. Протестированон на облачном deepseek-reasoner и Nemotron 3 Super   
Используется встроенный в openclaw телеграм бот
