# TOOLS

- **gog** – доступ к Google Sheets.  
  Команды: `gog sheets get <id> <диапазон>`, `gog sheets update <id> <ячейка> <значение>`.  
  Для получения всей таблицы используй `gog sheets get $GOG_TABLE A:Z --json` (если поддерживается).  
- **Скрипты** – при расчётах пиши временный Python‑скрипт в `/tmp` или в папку `scripts/`. Используй `gog` из скрипта через subprocess или прямой API.  
- **Память** – работай с файлами `memory/`, `MEMORY.md` (только в личке).  
- **CRON** – если есть `CRON.md`, следуй его инструкциям для периодических проверок.  
- **HEARTBEAT.md** – используй для планирования рутинных проверок (почта, календарь — по желанию).

При работе с таблицей всегда учитывай, что строки идут по датам. Недостающие даты пропускай, не додумывай.
